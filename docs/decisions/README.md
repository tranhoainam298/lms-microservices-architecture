# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the LMS system.

## Records

- `AD01-microservices.md` - Decision to use Microservices Architecture
- `AD02-api-gateway.md` - Decision to use API Gateway pattern
- `AD03-database-per-service.md` - Decision to use Database per Service pattern
- `AD04-message-broker.md` - Decision to use Message Broker for event-driven communication
- `AD05-external-services.md` - Decision on external service integration (ZaloPay/Momo, AI Chatbot)
