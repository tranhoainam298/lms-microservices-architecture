# Learning Management System - Microservices Architecture

This project is a scaffold for the LMS architecture project.

Current phase:
- Folder skeleton only
- No implementation code yet
- No database schema yet
- No Docker configuration yet

Architecture:
- Microservices Architecture
- API Gateway
- Event-driven communication through Message Broker
- Database per Service

Services:
- User Service
- Course Service
- Exam & Quiz Service
- Payment Service

Databases:
- User DB
- Course DB
- Exam DB
- Payment DB

External Systems:
- Payment Gateway ZaloPay/Momo
- AI Chatbot System

Do not create services or databases outside the architecture document.
